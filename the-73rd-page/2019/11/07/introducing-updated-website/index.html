Introducing Updated Website

Christopher Tobin

November 07, 2019

I had been rolling with the Minimal Mistakes theme for quite a while on this website. It served me well in its time here. I really loved the design - it looked gorgeous. But I never really liked how much had to be configured in the front matter in order to get certain things working. Anyway, I’m ready for a change.

I ended up using the Minima theme as a bit of a stopgap while I figured out what I wanted to do next. I waffled back and forth trying to decide whether I wanted to use an existing theme.

I ultimately ended up deciding to not use a theme. I found a lot of the themes a bit too restrictive. If I wanted to do something as simple as adding customer items to the page <head> or add some different items to the page footer, I had to override these entire sections. What’s the point in using a theme if I would have to override large sections of the theme to add some pieces of information?

By not using a theme, I was able to make the site do whatever I wanted. I’m pretty happy with how it turned out. I’ll be making some tweaks here and there over the coming days/weeks/months of course. Nothing is ever static in web development.
